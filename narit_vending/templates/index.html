    Operating Notes

    Hold-to-Run Test Jog

    1. Press ARM TEST MODE once before using Test Jog.
    2. Press and hold X±, Y± or Z± to rotate continuously.
    3. Release the Jog button to stop normal test motion.
    4. Use the physical E-Stop immediately for an unsafe response.
    5. Cancel Test Mode before returning to normal motion.
